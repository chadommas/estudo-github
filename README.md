# projeto-site
Projeto de um site criado durante o curso de Git e GitHub

## Processo de desenvolvimento

Este site foi desenvolvido como parte do meu aprendizado em Git, GitHub e GitHub Pages.

Durante o processo, criei branches separadas para testar novas páginas e estilos, implementei navegação entre múltiplas páginas HTML e publiquei o projeto utilizando GitHub Pages. Também enfrentei conflitos de merge envolvendo arquivos HTML e CSS, que foram resolvidos manualmente após análise das diferenças entre as versões.

As principais dificuldades envolveram o entendimento do funcionamento das branches, a resolução de conflitos e a organização de caminhos relativos para imagens e links entre páginas. Esses desafios contribuíram significativamente para o meu entendimento prático de versionamento, deploy e estrutura de projetos web estáticos.
